Trailer Gremlins

el atributo href puede adoptar tres formas distintas:

  • Vinculos relativos: como las imagenes, tienen una ruta usando como referencia el lugar donde se encuentra el documento .html
  • Vinculo Absoluto: esto es lo mismo que con laas imagenes. podemos usar una url completa. siempre teniendo en cuenta no usar un link de busqueda sino la url concreta del recurso a vincuñlar
  • Vinculo Ancla: los vinculooos ancla se usan para referenciar a las distintas partes del mismo documento. para utilizarlo se deben usar el atributo global id que debe ser unico en el documento

por otro lado los vinculos tienen otro atributo llamado target

el atributo target junto con rel definen como se va comportar el vinculo, en este caso usando el valor _blank nos abre una nueva pestaña y sin referencias a la pagina actual
